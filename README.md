# 🚀 IGNITO 2026 – Space-Themed Techfest Website

IGNITO 2026 is a futuristic, space-inspired techfest website designed to showcase events, competitions, workshops, and festival information through an immersive user experience.

The project combines modern UI design, smooth animations, interactive elements, and responsive layouts to create a visually engaging digital experience.

## ✨ Features

- 🌌 Space-themed immersive landing page
- 🚀 Interactive hero section
- ⭐ Animated starfield background
- 🧑‍🚀 Floating astronaut animation
- 🪐 Orbital visual elements and trajectories
- ⏳ Live countdown timer
- 🎯 Events & Competitions showcase
- 📱 Fully responsive design
- 🎨 Modern glassmorphism-inspired UI
- ⚡ Smooth scrolling and transitions
